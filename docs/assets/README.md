# Assets (images and screenshots)

Place photographs and screenshots referenced by the documentation in this
folder, then replace the corresponding placeholder lines in the pages.

Placeholders look like this in the Markdown:

```
📷 *Screenshot placeholder: QU-5D power switch location.*
```

Replace with a real image, for example:

```
![QU-5D power switch](../assets/qu5d-power-switch.jpg)
```

## Suggested images to capture

- Wall power switches (labelled)
- QU-5D mixer top view with faders/mutes labelled
- QU-5D power switch and Scenes screen
- JBL SRX812P speaker power switch; foyer amplifier
- NUC power button and StreamDeck XL lit up
- StreamDeck XL full layout with button labels
- RodeCaster Video front panel (Stream button, meters)
- AVKANS Joy Pro Controller (joystick, zoom, camera-select)
- Handheld mic battery compartment; headset bodypack battery door
- Bluestream matrix control page / remote
- TV input/source selection on the remote

## Naming

Use lower-case, hyphenated names that match the device, e.g.
`rodecaster-stream-button.jpg`, `streamdeck-layout.png`.
